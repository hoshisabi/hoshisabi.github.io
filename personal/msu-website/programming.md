# Programming STUFF! — Files of Interest

*Archived from http://www.msu.edu/~chapmand/program.html (captured Feb 18, 2005 via Wayback Machine)*

---

## Files

| Filename | Description |
|----------|-------------|
| ipxstart.zip | A starter kit for how to program IPX (networked) applications |
| inifile.zip | A kit for reading/writing .INI files from DOS |
| vgafonts.zip | A library for reading & using text mode VGA fonts |
| pcpge10.zip | PC Game Programmer's Encyclopedia. Good reference on a variety of topics |
| patch10a.zip | Patch to pcpge10.zip (corrects several mistakes) |
| wpcgpe10.zip | PCGPE in Windows Help format |
| ipxchat.zip | IPX Chat program (beta test of my IPX library) — see [ipxchat.md](ipxchat.md) |

*Note: The .zip files were hosted at http://www.msu.edu/~chapmand/files/ and are unlikely to be recoverable from the archive.*
