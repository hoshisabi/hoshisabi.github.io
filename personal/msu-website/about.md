This is the four hundred sixth rendition of this page; (Give or take four hundred) From that statement, you can probably gather that I'm the type that likes to play with computers for no good reason. My goal is to eventually be omniscient in the field of computers. I estimate that my goal shall be accomplished sometime in the year 3040 provided that computers freeze in development. I think it's a realistic goal.

I currently work for The MEDSTAT Group. I now program in Visual BASIC and Visual C++ (much more BASIC than C++). I used to do FORTRAN programming for them, but who am I to complain about transfers?

I'm a Wizard on a MUD named Vincent's Hollow. I also am a GM for an ongoing Shadowrun game. I used to run an Advanced Dungeons and Dragons campaign set in Greyhawk with Planescape crossovers. I've been trying hard to get my group to try out Ars Magica. I have also run many other gaming systems, which I'll create a separate page for later. Like all geeks, I also play Magic: The Gathering, Legend of the Five Rings and video games, as well.

In case anyone was curious, my system is a P5/133 with 24 megs of RAM, a nice video card and a dying monitor, running Win98, and a ton of hard disks in the thing. Occasionally, I'll try to make a game with it using Watcom C++, Turbo C++ or Visual C++, but I'll run into something more interesting on the way *(like this page)*, and end up getting sidetracked. I have, however, made the communications library for it finally. It supports both Serial (modem) and IPX (network) communications. So, if I ever do finish it, I should be able to get it to multi-play through DOS.
