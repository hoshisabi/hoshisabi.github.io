# IPX Chat — Beta Testers

*Archived from http://www.msu.edu/~chapmand/ipxbeta.shtml (captured Feb 18, 2005 via Wayback Machine)*  
*Copyright 1998 by Code Poets. Last revised: September 28, 1999.*

---

A DOS/Windows chat program built on a custom IPX networking library — one of the programming projects from [program.html](programming.md). The `ipxstart.zip` starter kit on the programming page was the underlying library; this was a working application built on top of it.

Beta testers signed up via a form collecting email, name, network type, and OS. The form submitted to an MSU CGI endpoint and routed to chapmand@concentric.net.

The download link (`ipxchat.zip`) was also referenced from the old index page. IPX networking was a Novell protocol used for LAN gaming before TCP/IP became universal — this was pre-internet-gaming infrastructure, for local network play.
