# Kittydance

*Archived from http://www.msu.edu/~chapmand/kittydance.html (captured Feb 18, 2005 via Wayback Machine)*

---

"Yeh, yeh, lame lame lame"

A Hamster Dance parody: a grid of animated cat GIFs (acat1.gif) alternating with "kismet.gif" (apparently a cat named Kismet), arranged in a diagonal-wave pattern across six rows. The page played GETDOWN.wav on load.

The nav button description said "Sabby is trying to be nouveau cool" — this, alongside [nouveau.html](nouveau.md), were the two pages filed under that category.
